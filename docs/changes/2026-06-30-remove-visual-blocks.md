# 2026-06-30 删除页面展示块

## 变更目标

根据页面截图反馈，删除以下用户可见内容：

- 留资二维码卡片和“打开留资入口”按钮。
- 资源库页面中的资源卡片列表和“打开内容页”按钮。

- 不删除 `src/content/resources.json` 中的资源数据。

## 验证方式

- 命令验证：`npm run typecheck`、`npm run lint`、`npm run test`、`npm run build`。
